📌 Overview

Cookingo reimagines how people learn cooking.
Instead of long videos or static recipes, it breaks learning into interactive quizzes and game-like challenges.

Think learning cooking the way games teach skills:
fast feedback, progression, and rewards.

✨ Features
🎮 Gamified Learning
Arcade-style progression system
Levels, streaks, and rewards
🧠 Quiz-Based Learning
Learn ingredients, techniques, and recipes
Bite-sized, interactive questions
📈 Progress Tracking
Track skill growth over time
Unlock new levels as you improve
🔥 Engaging UX
Designed to feel like a game, not a course
Quick sessions, high retention focus
